= Usage

OpenPolicyAgent can be defined in a YAML file:

[source,yaml]
----
apiVersion: authz.stackable.tech/v1
kind: OpenPolicyAgent
metadata:
  name: simple-opacluster
spec:
  version: "0.27.1"
  servers:
    selectors:
      default:
        selector:
          matchLabels:
            kubernetes.io/hostname: "foobar"
        instances: 1
        instancesPerNode: 1
        config:
          port: 8181
          repoRuleReference: "http://foobar:3030/opa/v1"

----
